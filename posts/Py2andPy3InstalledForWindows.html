Windows同时安装python2和python3

发表于2021-12-08|更新于2021-12-13
|阅读量:

使用python开发时，环境有python2和python3两种，为了满足不同版本需求，我们可以在Windows上搭建类似于kali下两版本共存的生产环境。

1、下载

在官网下载两个不同版本的python安装程序

python3是后缀为exe的可执行文件

python2是后缀为mis的安装程序

下载地址：https://www.python.org/downloads

2、安装

2-1、安装python3

在安装时勾选Add Python 3.xx to PATH（添加到系统环境变量）后可以在CMD控制台中输入python直接运行，其余选项根据自身需求或选择Install Now默认即可

安装完成后在控制台输入python测试即可

2-2、安装Python2

Python2安装时无需添加到环境变量，我们后续手动添加，直接安装即可。

3、配置

3-1、Python3配置

打开python3安装目录找到scripts文件夹，删除文件夹内的pip.py

（python默认安装目录：C:\Users\用户名\AppData\Local\Programs\Python\Python3）

3-2、Python2配置

打开python2安装目录修改文件python.exe为python2.exe

打开python2安装目录找到scripts文件夹，删除文件夹内的pip.py

3-3、配置环境变量

  • 按照下图方式打开环境变量配置（此时以Win11为例，Win10同理）

  • 双击系统变量中的Path添加两条规则

（添加内容分别为python2安装目录和python2安装目录下的scripts文件夹）

4、测试

4-1：python测试

4-2：pip测试

5、总结

安装步骤：

  • 1、安装两个版本的python
  • 2、同时删除两个scripts下的pip
  • 3、修改python2目录下的python.exe文件为python2.exe
  • 4、配置python2环境变量

因为在执行pip命令时需要调用到python安装目录下的python.exe但是我们对python2的可执行文件进行了重命名所以在使用pip2时使用python2 -m pip install <Package>即可
